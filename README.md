# 🩺 Child Malnutrition Detection and Intervention System

A hybrid AI-based healthcare decision support system developed to detect child malnutrition, assess nutritional risk, and recommend intervention strategies using multiple Artificial Intelligence techniques.

## 🚀 Features

- Machine Learning models for malnutrition prediction and risk assessment
- Reinforcement Learning for optimal intervention planning
- Prolog-based expert system for explainable diagnosis
- A* and AO* search algorithms for treatment path optimization
- K-Means and Hierarchical Clustering for nutritional case analysis
- Flask-based web interface
- Dockerized microservice architecture

## 🛠️ Technologies Used

- Python
- Flask
- SWI-Prolog
- Scikit-learn
- Reinforcement Learning
- Docker & Docker Compose
- HTML, CSS, JavaScript

## 🏗️ Project Structure

```
ml_service/       → ML models and clustering
prolog_service/   → Expert system and search algorithms
rl_service/       → Reinforcement Learning module
orchestrator/     → Flask application and UI
```

## ▶️ Running the Project

```bash
docker-compose up --build
```

Then open:

```
http://localhost:5000
```
